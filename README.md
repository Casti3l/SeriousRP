#SeriousRP

Welcome on my first plugin. This plugin make you server more Rp than before, this add somes commands like random tp, or towns.
The plugin had 3 translations (English, French, and German), but you can add yours by sending the file "config.yml" translated at srp.feedback@stan1712.fr

##Default configuration
```
#+-- Plugin RP by stan1712 (stan1712.fr) --+
#+-- Traductors : stan1712 (French) | stan1712 (English) | legaming04 (German)

#Version of the plugin, please don"t change it !
Version: "Version 3.3"

#Modify your prefix here
Prefix: "SeriousRP"

#Messages of the help message
HelpMsg:
  Dsrtp: "Be able to teleport randomly"
  DVersion: "Know the version of the plugin"
  DHelp: "Show the help"
  DTown: "Be able to teleport in the town"
  DTownSet: "Be able to set the teleport point of the town"

#Differents messages of the plugin
Teleports:
  GoToTown: "You have been teleported in the town"
  SetTown: "You have set the teleport point of the town"
  GoToPort: "You have been teleported in the port"
  SetPort: "You have set the teleport point of the port"

RandomTeleport: "Teleportation has thousands of blocks"

Help: "Need help ? Type /seriousrp help"
VersionHelp: "Want to know the version of the plugin ? Type /seriousrp version"

Locations:
  Town:
    x: 0
    y: 0
    z: 0
    WorldName: world
```
