#### Commandes RP :
* /rtp : Permet de vous téléporter aléatoirement dans un rayon de 1000 blocks.

#### Commandes Système :
* /seriousrp version : Permet de voir la version du plugin.
* /seriousrp help : Permet d'afficher une page d'aide.
